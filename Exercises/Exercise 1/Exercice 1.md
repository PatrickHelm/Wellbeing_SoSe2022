## Programming Fundamentals

### Operators

#### Arithmetic Operators

| Command | Meaning                         |
|---------|---------------------------------|
| +       | Addition                        |
| -       | Subtration                      |
| *       | Multiplication                  |
| /       | Division                        |
| ^ or ** | Exponentiation                  |
| %/%     | Whole Number Division           |
| %%      | Modulo (Remainder of a Division)|

#### Logic Operators

| Command | Meaning                         |
|---------|---------------------------------|
| ==      | Equal                           |
| !=      | Unequal                         |
| <       | Smaller than                    |
| >       | Greater than                    |
| <=      | Smaller equal                   |
| >=      | Greater equal                   |
| &       | Logical AND                     |
| &#124;  | Logical OR                      |
| !       | Logical NOT                     |
